## 前言

这是一个来自书《锋利的jQuery(第2版)》的demo，有两个网页。

## 功能

网站首页实现的功能有：
```
* 搜索框文字效果
* 网页换肤
* 导航效果
* 左侧商品分类热销效果
* 中间大屏广告效果
* 右侧最新动态模块内容添加超链接提示
* 右侧下部鼠标滑过产品列表效果
```
